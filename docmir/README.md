# Usage
<!-- usage -->
```sh-session
$ npm install -g docker-mirror
$ docmir COMMAND
running command...
$ docmir (-v|--version|version)
docker-mirror/0.0.0 darwin-x64 node-v16.0.0
$ docmir --help [COMMAND]
USAGE
  $ docmir COMMAND
...
```
